# 🖥️ SOC Simulation

## 📌 Description

A Python project that simulates a Security Operations Center (SOC) environment by generating and analyzing security logs.

## 🎯 Objective

Provide a practical understanding of how security analysts monitor and detect suspicious activities.

## ⚙️ How it works

* Generates simulated log events
* Identifies suspicious patterns (e.g., brute force, abnormal access)
* Outputs alerts for analysis

## ▶️ How to run

```bash
python main.py
```

## 📊 Example Output

[ALERT] Multiple failed login attempts detected
[WARNING] Suspicious IP activity detected

## 🧠 Security Context

Simulates real SOC workflows, including monitoring, detection, and basic incident analysis.
